# **Vaii Semestrálka**

`Vue.js` + `Express`

## **Zapnutie**

### **Front-End časť**

Front-End časť sa nachádza v priečinku `vaii-frontend`
```bash
npm install
npm run dev
```

### **Back-End časť** *(Nedokončené)*

Back-End časť sa nachádza v priečinku `vaii-backend`
```bash
npm install
npm run dev
```
